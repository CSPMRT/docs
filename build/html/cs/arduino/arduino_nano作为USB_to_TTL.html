文档元数据

作者：Chen Longhao

创建时间：2021年3月27日

硬件部分修改¶

将nano的 reset 与地短接，其他部分不作修改。

软件部分修改¶

将以下程序写入用作USB to TTL的nano中

void setup()
{
        pinMode(0, INPUT_PULLUP);
        pinMode(1, INPUT_PULLUP);
}

void loop() {}
