\(p+p\stackrel{}{\longrightarrow}3p+\overline{p}\) 阈值能量计算¶

文档元数据

作者：Chen Longhao

创建时间：2021年5月24日

设质子静止质量为 \(m_p\) ，运动的质子的四维动量为 \(P_1\) ，静止的为 \(P_2\) 。 要使其碰撞所需的能量最低，则产生的3个质子和反质子在质心坐标系中静止。 故，设产生的质子和反质子的四维动量为 \(P_3\) ，则在质心坐标系中有：

\[P_3 = (m_pc, 0, 0, 0)\]

根据动量守恒，有

\[P_1 + P_2 = 4P_3\]

因为四维矢量的平方在洛伦兹变换中是不变量，可以方便的进行坐标系变换，所以我们两边平方得到

\[ \begin{align}\begin{aligned}(P_1 + P_2)^2 = (4P_3)^2\\P_1^2 + 2P_1P_2 + P_2^2 = (4P_3)^2 = 16m_p^2c^2\end{aligned}\end{align} \]

又

\[ \begin{align}\begin{aligned}P_1^2 = m_p^2c^2\\P_2^2 = m_p^2c^2\end{aligned}\end{align} \]

故

\[P_1P_2 = 7m_p^2c^2\]

因为在实验室坐标系中：

\[ \begin{align}\begin{aligned}P_1 = (\frac{E}{c}, p, 0, 0)\\P_2 = (m_pc, 0, 0, 0)\end{aligned}\end{align} \]

所以

\[E = 7m_pc^2\]

此能量包含了质子静能 \(m_pc^2\) ，所以阈值能量为 \(6m_pc^2\)
